## 実装計画書: サービス名 Autocomplete 機能

## 目的

`ServiceModal` のサービス名入力欄に、`simple-icons` の全ブランドを候補ソースとした入力補助 (Autocomplete) を追加する。**自由入力を阻害せず、ヒント表示のみ**。

## 前提

- 既存ファイル
  - `src/components/dashboard/ServiceModal.tsx` — サービス名 `<Input>` を含むモーダル
  - `src/components/dashboard/ServiceIcon.tsx` — ブランドアイコン描画
  - `src/lib/icon-map.ts` — 名前 → SimpleIcon の手書き辞書 (**今回は変更しない**)
- 使用パッケージ: `simple-icons` (既にインストール済み、`icons.json` を同梱)
- 既存スタック: Next.js 16 (App Router) / React 19 / Tailwind / shadcn/ui / Biome / pnpm

## 仕様

### マッチング仕様

- 入力が **2 文字以上** で候補表示開始
- マッチ対象: `title`、`slug`、`aliases.aka` (それぞれ lowercase)
- スコアリング (高い順):
  1. `title` が prefix 一致
  2. `slug` が prefix 一致
  3. `aka` のいずれかが prefix 一致
  4. `title` / `slug` / `aka` の部分一致 (`includes`)
- 上位 **8 件** まで表示
- 同じ slug は重複させない (aka 経由でヒットしても 1 件)

### UI 仕様

- 入力欄は既存 `<Input>` を残し、**直下に候補リストを絶対配置** (Popover 不使用)
- 候補が 0 件、または入力 2 文字未満、または focus 外なら非表示
- 候補項目: 左にブランドアイコン (24x24, hex で着色)、右に `title` 表示
- 候補クリックで input の値を `title` に置換し、リストを閉じる
- **Esc キー** / **外クリック** でリストを閉じる
- **矢印キー** ↑↓ で候補にフォーカス移動、**Enter で確定**。候補にフォーカスがない状態の Enter は通常の form submit を阻害しない
- 自由入力は常に有効。候補を選ばずに任意の文字列を保存できる

## ステップ詳細

### Step 1: ビルドスクリプト `scripts/build-icon-index.ts`

**目的**: `node_modules/simple-icons/icons.json` を読み、Autocomplete に必要な最小フィールドだけに削った TypeScript ファイルを `src/lib/icon-index.generated.ts` に出力する。

**処理内容**:

```ts
// scripts/build-icon-index.ts
import { readFileSync, writeFileSync } from 'node:fs';
import { resolve } from 'node:path';

type RawIcon = {
  title: string;
  hex: string;
  slug?: string;
  aliases?: { aka?: string[] };
};

function toSlug(title: string): string {
  // simple-icons の slug 生成規則の簡易版: 小文字化し非英数を除去
  return title.toLowerCase().replace(/[^a-z0-9]/g, '');
}

const iconsPath = resolve(process.cwd(), 'node_modules/simple-icons/icons.json');
const raw = JSON.parse(readFileSync(iconsPath, 'utf-8')) as RawIcon[];

const index = raw.map((icon) => ({
  title: icon.title,
  slug: icon.slug ?? toSlug(icon.title),
  hex: icon.hex,
  aka: icon.aliases?.aka,
}));

const outPath = resolve(process.cwd(), 'src/lib/icon-index.generated.ts');
const content = `// AUTO-GENERATED by scripts/build-icon-index.ts. Do not edit by hand.
export type IconIndexEntry = {
  title: string;
  slug: string;
  hex: string;
  aka?: string[];
};

export const ICON_INDEX: readonly IconIndexEntry[] = ${JSON.stringify(index, null, 2)};
`;

writeFileSync(outPath, content, 'utf-8');
console.log(`Wrote ${index.length} icons to ${outPath}`);
```

**実行確認**: `pnpm tsx scripts/build-icon-index.ts` で `src/lib/icon-index.generated.ts` が生成され、3400+ エントリを含むこと。

注意: `tsx` が未導入なら `pnpm add -D tsx` を**ユーザーに提案**する (自分では実行しない)。

### Step 4: `src/lib/icon-suggest.ts`

**目的**: クエリ文字列から候補配列を返すピュア関数。

```ts
import { ICON_INDEX, type IconIndexEntry } from './icon-index.generated';

export type Suggestion = {
  title: string;
  slug: string;
  hex: string;
};

const MIN_QUERY_LENGTH = 2;
const MAX_RESULTS = 8;

export function suggestServices(query: string): Suggestion[] {
  const q = query.trim().toLowerCase();
  if (q.length < MIN_QUERY_LENGTH) return [];

  type Scored = { entry: IconIndexEntry; score: number };
  const scored: Scored[] = [];

  for (const entry of ICON_INDEX) {
    const title = entry.title.toLowerCase();
    const slug = entry.slug.toLowerCase();
    const akaList = entry.aka?.map((a) => a.toLowerCase()) ?? [];

    let score = 0;
    if (title.startsWith(q)) score = 100;
    else if (slug.startsWith(q)) score = 90;
    else if (akaList.some((a) => a.startsWith(q))) score = 80;
    else if (title.includes(q)) score = 50;
    else if (slug.includes(q)) score = 40;
    else if (akaList.some((a) => a.includes(q))) score = 30;

    if (score > 0) scored.push({ entry, score });
  }

  scored.sort((a, b) => {
    if (b.score !== a.score) return b.score - a.score;
    return a.entry.title.localeCompare(b.entry.title);
  });

  return scored.slice(0, MAX_RESULTS).map(({ entry }) => ({
    title: entry.title,
    slug: entry.slug,
    hex: entry.hex,
  }));
}
```

### Step 5: `src/components/dashboard/ServiceNameAutocomplete.tsx`

**目的**: 既存 `<Input>` をラップし、Autocomplete UI を付与する Client Component。

**Props**: 既存 `Input` と互換にする。最低限:

```ts
type Props = {
  id?: string;
  name?: string;
  value: string;
  onChange: (value: string) => void;
  required?: boolean;
  placeholder?: string;
};
```

**実装方針**:

```tsx
'use client';

import { useEffect, useRef, useState } from 'react';
import { Input } from '@/components/ui/input';
import { suggestServices, type Suggestion } from '@/lib/icon-suggest';

type Props = {
  id?: string;
  name?: string;
  value: string;
  onChange: (value: string) => void;
  required?: boolean;
  placeholder?: string;
};

export function ServiceNameAutocomplete(props: Props) {
  const [open, setOpen] = useState(false);
  const [activeIndex, setActiveIndex] = useState(-1);
  const containerRef = useRef<HTMLDivElement>(null);

  const suggestions: Suggestion[] = open ? suggestServices(props.value) : [];

  // 外クリックで閉じる
  useEffect(() => {
    function handleClickOutside(e: MouseEvent) {
      if (containerRef.current && !containerRef.current.contains(e.target as Node)) {
        setOpen(false);
      }
    }
    document.addEventListener('mousedown', handleClickOutside);
    return () => document.removeEventListener('mousedown', handleClickOutside);
  }, []);

  function handleSelect(s: Suggestion) {
    props.onChange(s.title);
    setOpen(false);
    setActiveIndex(-1);
  }

  function handleKeyDown(e: React.KeyboardEvent<HTMLInputElement>) {
    if (!open || suggestions.length === 0) return;

    if (e.key === 'ArrowDown') {
      e.preventDefault();
      setActiveIndex((i) => (i + 1) % suggestions.length);
    } else if (e.key === 'ArrowUp') {
      e.preventDefault();
      setActiveIndex((i) => (i <= 0 ? suggestions.length - 1 : i - 1));
    } else if (e.key === 'Enter' && activeIndex >= 0) {
      e.preventDefault();
      handleSelect(suggestions[activeIndex]);
    } else if (e.key === 'Escape') {
      setOpen(false);
      setActiveIndex(-1);
    }
  }

  return (
    <div ref={containerRef} className="relative">
      <Input
        id={props.id}
        name={props.name}
        value={props.value}
        onChange={(e) => {
          props.onChange(e.target.value);
          setOpen(true);
          setActiveIndex(-1);
        }}
        onFocus={() => setOpen(true)}
        onKeyDown={handleKeyDown}
        required={props.required}
        placeholder={props.placeholder}
        autoComplete="off"
      />
      {open && suggestions.length > 0 && (
        <ul
          className="absolute left-0 right-0 top-full z-50 mt-1 max-h-72 overflow-auto rounded-md border border-stone-200 bg-white shadow-md"
          role="listbox"
        >
          {suggestions.map((s, i) => (
            <li key={s.slug}>
              <button
                type="button"
                role="option"
                aria-selected={i === activeIndex}
                className={`flex w-full items-center gap-2 px-3 py-2 text-left text-sm hover:bg-stone-100 ${
                  i === activeIndex ? 'bg-stone-100' : ''
                }`}
                onMouseDown={(e) => e.preventDefault()}
                onClick={() => handleSelect(s)}
              >
                <span
                  className="inline-flex h-5 w-5 shrink-0 items-center justify-center"
                  style={{ color: `#${s.hex}` }}
                >
                  {/* アイコンは ICON_INDEX に path を持たせていないため、ここではブランドカラーの丸印で代用。
                     もし path も欲しい場合は icon-index 側に path を追加する */}
                  <span className="h-3 w-3 rounded-full" style={{ background: `#${s.hex}` }} />
                </span>
                <span className="truncate">{s.title}</span>
              </button>
            </li>
          ))}
        </ul>
      )}
    </div>
  );
}
```

**設計注**:
- `onMouseDown={(e) => e.preventDefault()}` は input の blur より先にクリックを成立させるため
- 候補の左マークは「ブランドカラーの小さな丸」で代用 (`ICON_INDEX` に `path` を含めると JSON が肥大化するため)。もし**実アイコン表示が必要**なら、`icon-index.generated.ts` に `path` フィールドも含めて `ServiceIcon` の描画ロジックを再利用する。**実装着手前に確認**

### Step 6: `ServiceModal.tsx` の差し替え

`ServiceModal.tsx` のサービス名 input を特定する:

```bash
grep -n 'name="name"' src/components/dashboard/ServiceModal.tsx
```

該当 `<Input>` を `<ServiceNameAutocomplete>` に置き換える。state 管理が `useState` なのか `formData` 直渡しなのかで `value` / `onChange` の繋ぎ方が変わるので、**既存実装を Read してから差し替える**。

想定パターン:
- 既に `useState` 管理 → そのまま `value` / `onChange` を渡す
- uncontrolled (formData ベース) → `ServiceNameAutocomplete` を controlled で導入するため、ローカルに `useState` を追加し、`name` 属性で form submit に乗せる
